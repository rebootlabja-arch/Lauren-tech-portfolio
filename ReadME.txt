# Expense Data Analysis Project

## Overview

This project analyzes expense data using Python to identify spending patterns and generate insights.

## Tools & Technologies

* Python (pandas, matplotlib)
* Jupyter Notebook
* Microsoft Excel

## What I Did

* Cleaned and structured raw data
* Converted data types for analysis
* Analyzed spending by category and date
* Created visualizations to highlight trends

## Key Insights

* Highest spending category: Food
* Highest spending day: April 5, 2026
* Most frequent transactions: Food category
* Spending varies across different days

This project demonstrates my ability to:

* Work with real-world data
* Clean and prepare datasets
* Perform data analysis using Python
* Communicate insights clearly

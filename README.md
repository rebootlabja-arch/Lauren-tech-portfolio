# Expense Data Analysis

## Summary

This project focuses on analyzing personal expense data to identify spending patterns, trends, and insights using Python and Excel.

---

## Objective

The goal of this project is to:

* Understand how to work with real-world data
* Clean and prepare data for analysis
* Extract meaningful insights from raw data
* Present findings using visualizations

---

## Tools & Technologies

* Python (pandas, matplotlib)
* Jupyter Notebook
* Microsoft Excel

---

## Dataset

The dataset contains records of daily expenses, including:

* Date of transaction
* Item purchased
* Category (Food, Transport, Bills)
* Cost

---

## Data Cleaning Process

* Converted the Date column into proper datetime format
* Standardized category values (removed inconsistencies)
* Converted Cost column to numeric format
* Handled missing values appropriately

---

## Analysis Performed

* Total spending calculation
* Spending by category
* Spending by date
* Transaction frequency by category
* Spending trends by day of the week

---

## Key Insights

* Food is the highest spending category
* April 5, 2026 recorded the highest total spending
* Food category has the highest number of transactions
* Spending varies significantly across different days

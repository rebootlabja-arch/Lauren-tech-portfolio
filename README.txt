# Smart Utility Billing System

A Python-based console application designed to manage customer electricity billing records efficiently.

## Features

* Add and manage customer bills
* Automatically calculate subtotal, tax, and total charges
* Generate formatted electricity receipts
* Search customer records
* View complete bill history
* Export billing data to CSV files
* Menu-driven user interface

## Technologies Used

* Python
* CSV File Handling
* Datetime Module

## Programming Concepts Applied

* Functions
* Dictionaries and Lists
* Loops and Conditional Statements
* File Handling
* Data Organization

## Purpose

This project was built to strengthen my Python programming skills through a practical real-world system. It helped improve my understanding of data management, program structure, and user interaction in console applications.
